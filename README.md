# kris-mars-rover

Kris's solution for a [Mars-Rover](https://github.com/abdulg/Mars-Rover)

## Instructions

Requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd kris-mars-rover
npm i
node app
```

#### Tests

To run tests:

```sh
cd kris-mars-rover
npm i
npm run test
```
